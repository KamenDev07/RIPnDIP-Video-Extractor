## Requirements
- [`yt-dlp`](https://github.com/yt-dlp/yt-dlp) (Python library)
- [`ffmpeg`](https://ffmpeg.org/download.html) (must be installed separately)

## Installation
Install the required Python packages:

```bash
pip install -r requirements.txt
```
Or install yt-dlp directly:
```bash
pip install yt-dlp
```
